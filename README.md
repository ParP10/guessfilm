В файле GuessFilm.java содержится главный класс проекта GuessFilm.
Существует 4 режима работы:
* Режим классификации (режим угадывания фильма)
* Режим тренировки
* Режим добавления вопросов в базу данных
* Режим добавления фильмов в базу данных

Класс Films в файле Films.java содержит данные и методы для работы со всеми фильмами.
Класс Film в файле Film.java содержит данные и методы для работы с одним фильмом.
Класс Questions в файле Questions.java содержит данные и методы для работы со всеми вопросами.
Класс Question в файле Question.java содержит данные и методы для работы с одним вопросом.
Класс DataBase в файле DataBase.java организует работу с базами данных.
Класс User в файле User.java организует взаимодействие с пользователем.
Класс Learning в файле Learning.java организует процесс обучения и классификации программы, используя библиотеку WEKA.

Общение с пользователем происходит через консоль, данные хранятся в текстовых файлах.

При запуске программы необходимо выбрать режим работы. При выборе режима классификации (основной режим), программа задает вопросы о фильме пользователю.
Варианты ответов:
"y" - да
"n" - нет
"d" - не знаю
любой другой символ завершит работу программы.
После кадого полученного ответа пользователь видит результат классификации.

Вопросы задаются в заданном порядке.

В режиме обучения классификатор обучается на тренировочном корпусе (файл trainset.txt)
В режиме добавления вопросов в базу данных новые вопросы (находящиеся в файле newQuestions.txt) добавляются в файл questions.txt
В режиме добавления фильмов в базу данных новые фильмы (находящиеся в файле newFilms.txt) добавляются в файл films.txt
